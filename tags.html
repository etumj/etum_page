Important HTML tags

This page is designed as a quick reference for some of the common tags covered in Learn Enough HTML to be dangerous . In the process of making it, we'll learn how to make HTML tables via table and other related tags.

The tables below don't include all HTML tags, but they do list many of the most important ones.

Block Elements

Tag Name Purpose
h1–h6 headings include a heading (levels 1–6)
p paragraph include a paragraph of text
table table include a table
tr table row make a table header
th Table Header make a table header
td table data include a table data cell

Inline Elements

Tag Name Purpose Example Result
em emphasized make emphasized text <em>technical sophistication</em> technical sophistication
strong Bold Bolden text <b>Boldening sophistication</b> boldened sophistication
a anchor To add links in a webpage <a href="www.cnn.com">cnn</> facebook
img image To add pictures in a webpage <img src="clear.jpg"/> facebook
